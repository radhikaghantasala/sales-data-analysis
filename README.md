# 📊 Sales Data Analysis using Python

## 📌 Project Overview
This project focuses on analyzing monthly sales data using Python. It demonstrates key concepts of Data Science including data preprocessing, exploratory data analysis (EDA), visualization, and statistical analysis.

## 🎯 Objectives
- Understand sales trends over time
- Analyze the relationship between sales and profit
- Perform statistical analysis for better insights
- Visualize data using graphs and charts

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
